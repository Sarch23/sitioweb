¿Qué es JavaScript?

Definición

JavaScript es un lenguaje de programación interpretado que se ejecuta en el navegador del cliente. Es uno de los pilares fundamentales de la web moderna junto con HTML y CSS.

JavaScript permite crear páginas web interactivas dinámicas. Se puede integrar en páginas HTML para responder a eventos del usuario, manipular el DOM, enviar y recibir datos, y mucho más.

Características Principales

  • Lenguaje de alto nivel
  • Interpretado (no requiere compilación previa)
  • Dinámico (tipado débil y dinámico)
  • Multi-paradigma (orientado a objetos, funcional, imperativo)
  • Basado en prototipos
// Ejemplo simple de JavaScript
let mensaje = "¡Hola, mundo!";
console.log(mensaje); // Imprime: ¡Hola, mundo!

Tipos de Datos

Tipos Primitivos

Los tipos primitivos son inmutables y se pasan por valor. En JavaScript existen 7 tipos primitivos:

  • String
  • Number
  • BigInt
  • Boolean
  • Undefined
  • Null
  • Symbol

Tipos de Objeto

Los objetos son tipos complejos que pueden contener múltiples valores y métodos:

  • Object
  • Array
  • Function
  • Date
  • RegExp
  • Error
// Ejemplos de tipos primitivos
let cadena = "Hola"; // String
let numero = 42; // Number
let booleano = true; // Boolean
let nulo = null; // Null
let indefinido; // Undefined

// Ejemplo de objeto
let persona = {
  nombre: "Ana",
  edad: 30
}; // Object

Variables y Constantes

Declaración de Variables

En JavaScript moderno existen 3 formas de declarar variables:

  • var: Tiene alcance de función o global. No se recomienda usarlo
  • let: Tiene alcance de bloque. Se puede reasignar
  • const: Tiene alcance de bloque. No se puede reasignar
// Declaración con let (variable)
let contador = 0;
contador = 10; // Válido: se puede reasignar

// Declaración con const (constante)
const PI = 3.14159;
// PI = 3.14; // Error: no se puede reasignar

// Nota: para objetos y arrays, el contenido puede modificarse
const persona = { nombre: "Juan" };
persona.nombre = "Pedro"; // Válido: se modifica el contenido, no la referencia

Operadores

Clasificación de Operadores

Los operadores permiten realizar operaciones con valores. Se clasifican en:

  • Aritméticos: +, -, , /, %, *
  • Asignación: =, +=, -=, *=, /=, %=
  • Comparación: ==, ===, !=, !==, >, <, >=, <=
  • Lógicos: &&, ||, !
  • Unarios: typeof, +, -, !, ++, --
  • Ternario: ? :
// Operadores aritméticos
let a = 10;
let b = 3;
let suma = a + b; // 13
let modulo = a % b; // 1

// Operadores de comparación
let igualdad = a == "10"; // true (coerción)
let identidad = a === "10"; // false (sin coerción)

// Operadores lógicos
let and = (a > 5) && (b < 5); // true
let or = (a > 15) || (b < 5); // true

Estructuras de Control

Estructuras Condicionales

Permiten ejecutar código basado en condiciones:

  • if/else: Evaluación condicional
  • switch: Evaluación de múltiples casos

Estructuras de Repetición

Permiten repetir bloques de código:

  • for: Bucle con contador
  • while: Bucle condicional
  • do...while: Bucle que se ejecuta al menos una vez
  • for...in: Recorre propiedades de objetos
  • for...of: Recorre elementos de colecciones
// Estructura condicional: if/else
let edad = 18;
if (edad >= 18) {
  console.log("Mayor de edad");
} else {
  console.log("Menor de edad");
}

// Bucle for
for (let i = 0; i < 5; i++) {
  console.log("Iteración " + i);
}

// Bucle while
let contador = 0;
while (contador < 3) {
  console.log("Contador: " + contador);
  contador++;
}

Funciones

Definición de Funciones

Las funciones son bloques de código reutilizables que realizan una tarea específica:

  • Funciones declaradas: function nombre() {}
  • Funciones expresadas: const nombre = function() {}
  • Funciones flecha: const nombre = () => {}
// Función declarada
function saludar(nombre) {
  return "¡Hola, " + nombre + "!";
}

// Función expresada
const sumar = function(a, b) {
  return a + b;
};

// Función flecha
const multiplicar = (x, y) => x * y;

// Llamada a funciones
console.log(saludar("Ana")); // "¡Hola, Ana!"
console.log(sumar(5, 3)); // 8
console.log(multiplicar(4, 7)); // 28

Objetos

Objetos en JavaScript

Los objetos son colecciones de propiedades que pueden contener valores y métodos:

  • Propiedades: Clave-valor (datos)
  • Métodos: Funciones dentro de un objeto
  • Creación: Literales, constructores, clases
// Objeto literal
const coche = {
  marca: "Toyota",
  modelo: "Corolla",
  anio: 2020,
  encendido: false,

  arrancar: function() {
    this.encendido = true;
    return "Coche encendido";
  },

  detener: function() {
    this.encendido = false;
    return "Coche apagado";
  }
};

// Acceso a propiedades
console.log(coche.marca); // "Toyota"
console.log(coche["modelo"]); // "Corolla"

// Llamada a métodos
console.log(coche.arrancar()); // "Coche encendido"

Arrays

Arrays en JavaScript

Los arrays son colecciones ordenadas de valores. Pueden contener cualquier tipo de dato:

  • Creación: [] o new Array()
  • Acceso: array[índice]
  • Métodos comunes: push(), pop(), shift(), unshift(), slice(), splice(), map(), filter(), reduce()
// Creación de arrays
const frutas = ["manzana", "banana", "naranja"];
const numeros = [1, 2, 3, 4, 5];

// Agregar elementos
frutas.push("uva"); // Agrega al final
frutas.unshift("mango"); // Agrega al inicio

// Eliminar elementos
frutas.pop(); // Elimina del final
frutas.shift(); // Elimina del inicio

// Transformar con map
const dobles = numeros.map(n => n * 2); // [2, 4, 6, 8, 10]

Métodos Comunes

Métodos de Strings

Los strings tienen métodos para manipular su contenido:

  • toUpperCase(), toLowerCase()
  • substring(), slice()
  • split(), trim()
  • replace(), indexOf(), includes()

Métodos de Arrays

Los arrays tienen métodos para manipular colecciones:

  • push(), pop(), shift(), unshift()
  • map(), filter(), reduce()
  • forEach(), find(), some(), every()
// Métodos de strings
let texto = " JavaScript es genial ";
console.log(texto.trim()); // "JavaScript es genial"
console.log(texto.toUpperCase()); // " JAVASCRIPT ES GENIAL "

// Métodos de arrays
const numeros = [1, 2, 3, 4, 5];
const pares = numeros.filter(n => n % 2 === 0); // [2, 4]
const suma = numeros.reduce((acc, n) => acc + n, 0); // 15

Entrada y Salida (I/O)

Entrada de Datos

Formas de obtener datos del usuario:

  • prompt(): Muestra un cuadro de diálogo para entrada de texto
  • Formularios HTML: Campos de entrada en la página

Salida de Datos

Formas de mostrar resultados:

  • alert(): Muestra un mensaje emergente
  • console.log(): Muestra en la consola del navegador
  • innerHTML: Modifica el contenido HTML de un elemento
// Entrada con prompt
let nombre = prompt("¿Cómo te llamas?");

// Salida con alert
alert("¡Hola, " + nombre + "!");

// Salida en consola
console.log("Usuario: " + nombre);

// Salida en página web
document.getElementById("resultado").innerHTML = "Bienvenido " + nombre;
